# Browser Css Selector

## Tag

```css
* {}
div {}
html {}
body {}
a {}
p {}
h1 {}
span {}
img {}
```

## Class

```css
.box {}
```

## ID

```css
#app {}
#video {}
```

## Attr

```css
[color=red]
```

## Pseudo Class

```css
selector:pseudo-class {}

&:hover {}
&:focus {}
&:active {}

&:nth-child(1) {}
```
