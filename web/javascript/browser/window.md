# window

常用整理

## 属性

```js
// 高度
window.innerHeight
// 宽度
window.innerWidth
// 顶层窗口
window.top
// 文档
window.document
// Location
window.location
// Navigator
window.navigator
// Screen
window.screen
// 存储
window.sessionStorage
window.localStorage
// 其他...
window.pageXOffset
window.pageYOffset
```

## 方法

```js
// 与帧刷新相关
window.requestAnimationFrame
window.requestIdleCallback
// 定时器
window.setTimeout
window.setInterval
```
